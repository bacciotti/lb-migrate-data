# LB Migrate Data
This plugin allows you to insert txt data into WP Database, based on categories and some rules.   


## Requirements / Tested on
![Wordpress 5.8.1](https://img.shields.io/badge/Wordpress-5.8.1-green)  
![PHP 7.4](https://img.shields.io/badge/PHP-7.4-blue)  


## Installation

- Download this repository
- Upload it to wp-content/plugins
- Activate it


## Usage
- Create a page or post with the following shortcode

`lbm_shortcode`

- Access the page or post created to run de script
